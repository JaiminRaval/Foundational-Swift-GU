# Swift Programming Learning Repository

Welcome to Foundational Swift programming learning repository! This repository is designed to help you learn and understand the foundational concepts of Swift programming language through a structured, week-by-week approach.

## About This Repository

This repository contains code examples, exercises, and few projects which are organized in a weekly format (week-00, week-01, etc.) to provide a progressive learning experience. 

Each week builds upon the previous one, ensuring a solid understanding of Swift fundamentals and advanced concepts.

Whether you're a complete beginner or looking to strengthen your Swift knowledge, this repository serves as a practical companion to your Swift learning journey.

## Learning Objectives

- Master Swift syntax and foundational concepts
- Understand object-oriented programming in Swift
- Learn protocol-oriented programming paradigms
- Build practical applications using Swift
- Develop best practices for iOS/macOS development
- Gain confidence in Swift development workflows
